FAQ

Pertanyaan yang sering ditanyakan.

Tentang Produk
Apa perbedaan Assertin dengan tools API Client lain?

Assertin fokus untuk kebutuhan Tester: Scenario Runner untuk menyusun flow E2E dan Performance Testing berbasis k6, semua dalam satu dashboard berbasis browser, tanpa instal aplikasi, dan gratis untuk mulai.

Apakah Assertin gratis digunakan?

Ya, kamu bisa mulai menggunakan Assertin secara gratis tanpa kartu kredit. Paket gratis memiliki beberapa limitasi dasar (jumlah workspace, collection, dan kapasitas performance test), dengan opsi upgrade untuk kebutuhan yang lebih besar.

Apakah data dan request saya aman?

Setiap akun memiliki workspace sendiri yang hanya bisa diakses setelah login. Data collection, environment, dan history tersimpan di database kami. Data tidak digunakan untuk tujuan selain menjalankan layanan dan tidak dijual ke pihak lain.

Konsep Testing
Apa itu API testing?

API testing adalah proses memverifikasi bahwa endpoint API mengembalikan response yang sesuai — status code, body, header, dan waktu respons — sesuai ekspektasi. Di Assertin, kamu bisa kirim request, lihat response, dan validasi hasilnya langsung dari browser. Pelajari lebih lanjut →

Apa itu performance testing?

Performance testing digunakan untuk mengukur seberapa cepat dan stabil sistem ketika menerima beban pengguna. Dengan Assertin, kamu bisa menjalankan load test berbasis k6 untuk melihat metrik seperti RPS, latency, dan error rate, sehingga bisa memastikan API kamu tetap responsif dan handal di bawah beban. Pelajari lebih lanjut →

Apa itu Scenario Runner?

Scenario Runner memungkinkan kamu menghubungkan beberapa request menjadi satu flow — misalnya login, ambil token, lalu hit endpoint lain — dan menjalankannya sebagai satu skenario E2E. Response dari request sebelumnya bisa dipakai sebagai input untuk request berikutnya.

Penggunaan
Bagaimana cara mulai belajar API testing?

Mulai dari memahami HTTP, request, response, dan status code. Lanjut dengan membuat test sederhana di Assertin sebelum masuk ke automation dan performance testing. Baca panduannya →

Bagaimana cara melakukan performance test?

Pilih collection yang sudah dibuat, atur jumlah virtual users, durasi, dan ramp up time, lalu jalankan. Assertin akan mengeksekusi load test berbasis k6 dan menampilkan hasil seperti RPS, latency p95, dan error rate.

Perlu install aplikasi atau plugin tambahan?

Tidak. Assertin berjalan penuh di browser — buka halaman, login atau coba sebagai guest, dan langsung bisa mulai mengirim request tanpa instalasi apa pun.

Apakah Assertin bisa dipakai untuk kerja tim?

Saat ini setiap akun memiliki workspace personal. Untuk kebutuhan tim, kamu tetap bisa membagikan hasil performance test dan riwayat request sebagai bukti ke anggota tim lain.

Apa itu API automation testing?

API automation testing adalah proses menjalankan pengujian API secara otomatis menggunakan skenario dan validasi agar pengujian bisa diulang dengan konsisten. Di Assertin, kamu bisa buat skenario dengan Scenario Runner, lalu jalankan kapan pun untuk memastikan API tetap bekerja sesuai ekspektasi. Pelajari lebih lanjut →
